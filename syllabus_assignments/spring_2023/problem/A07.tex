\problem Find the following antiderivative:
\[\int \frac{5x+9}{(x+5)(x-3)}~dx\]
\vspace{15mm}

\problem Find the following antiderivative:
\[\int \frac{x^3+2x^2-x+1}{(x+2)(x-1)}~dx\]
\vspace{15mm}

\problem Find the following antiderivative:
\[\int \frac{x+2}{x^2+4x-24}~dx\]
\vspace{15mm}


\problem Find the following antiderivative:
\[\int \frac{x+2}{x^3+x^2+x-3}~dx\]
(Hint: Note the denominator evaluates to $0$ when $x=1$.  This may help you with the factorization.)
\vspace{15mm}

\problem Find the following antiderivative:
\[\int \frac{x+2}{x^2+2x+2}~dx\]
(Hint 1: Complete the squares in the denominator.)

\noindent(Hint 2: One answer for the antiderivative would be $\frac{1}{2}\ln|x^2+2x+2| + \arctan(x+1) + C$)
